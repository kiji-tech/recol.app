## 言語に関する仕様

- **i18n-js** を使って管理する
- src/libs/i18n.ts ファイルに定義する
- src/languages/xx.jsonで各言語コードのテキストを管理する
- テキストを使う場合は**generateI18nMessageメソッド**を必ず使うこと
  - <Text>タグやToastなどの画面に表示するもの
- 特定のAPIで言語コードを使う場合は､i18n.localeを使用すること

## 言語ファイルに関する仕様

| 先頭    | 用途・説明                                                    |
| ------- | ------------------------------------------------------------- |
| FEATURE | 機能ごとのテキストを管理する                                  |
| COMMON  | OKなど､どの機能であっても共通的に利用されるテキストを管理する |
| MESSAGE | 主にサーバーから返却されるメッセージをコードごとに管理する    |
